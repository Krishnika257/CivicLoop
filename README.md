# CivicLoop — Hyperlocal Problem Solver

CivicLoop is an AI-powered civic issue reporting platform that helps citizens report problems, tracks them through escalation, and autonomously drafts formal letters to authorities.

## 🚀 Features

### Tier 1 — Core (Fully Implemented)
- **Photo-based issue reporting** with AI categorization (Gemini Vision)
- **Map view** with status-colored pins (Leaflet.js)
- **AI-generated precedent cards** on submission
- **Auto-escalation letters** with visual timeline (Day 0 / Day 7 / Day 14)
- **Resolution tracking** with manual status updates

### Tier 2 — Bonus
- **"Silent Majority"** mock detection (simulated)
- **Karma points** with mock business rewards
- **Community verification** (upvote/confirm)


## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | React + Vite + TypeScript |
| Styling | Tailwind CSS |
| Maps | Leaflet.js (OpenStreetMap) |
| Backend | Firebase (Auth, Firestore, Storage) |
| AI | Google Gemini API (1.5 Flash) |
| Hosting | Vercel (frontend) + Firebase (services) |
